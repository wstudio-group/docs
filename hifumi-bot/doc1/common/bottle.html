漂流瓶

扔漂流瓶

扔出一个漂流瓶

/扔漂流瓶
或
[回复消息] /扔漂流瓶

发送“取消”可以取消老师的漂流瓶操作！

捡漂流瓶

捡一个漂流瓶，这个漂流瓶可能来自其他群的老师或学生哦！

/捡漂流瓶

查看漂流瓶

查看指定序号的漂流瓶

/查看漂流瓶 n
（n为漂流瓶序号）

点赞

给某个漂流瓶点赞哦！
点赞后会通知那位老师的哦~

/点赞漂流瓶 n
（n为漂流瓶序号）

日富美懒得放图啦(///)~

评论

评价某位老师的漂流瓶！
评论后也是会通知那位老师的哦~

/评论漂流瓶 n [评论]
（n为漂流瓶序号）
（[评论]为老师需要评论的内容！）

日富美懒得放图啦(///)~

举报！

如果老师发现不好的内容，请立即举报给日富美的主人呀！
（举报五次会自动删除，并反馈给超管）

/举报漂流瓶 n
（n为漂流瓶序号）

日富美懒得放图啦(///)~

删除漂流瓶

将老师您曾经扔出去过的漂流瓶删除！

/删除漂流瓶 n
（n为漂流瓶序号）

日富美懒得放图啦(///)~

我的漂流瓶

看看老师您曾经扔过什么漂流瓶~

/我的漂流瓶
