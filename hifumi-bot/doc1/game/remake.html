人生重开

首先，开始游戏

@日富美 /人生重开

选择天赋

为你的重开选择天赋

n n n
（三个n对应选择的三个天赋的序号）
（发送“随机”可随机选择）

示例：0 1 2

分配属性

给每个属性分配点数

n n n n
（四个n对应四个属性分配的点数）
（发送“随机”可随机分配）
